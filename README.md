# Cryptonomicon

Практика. Работе с Vue 3. Работа с websocket. Учебный проект.

## Установка
```
npm install
```

### Компиляция и hot-reloads для Разработки
```
npm run serve
```

### Компиляция и минификация для production
```
npm run build
```

### Линтинг
```
npm run lint
```
